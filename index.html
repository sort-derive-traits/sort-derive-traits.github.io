Rust Derive Traits Sorter

What:

Easily organize your Rust derive traits for consistency.

Why:

Sorting derives keeps your code predictable and easy to read. While cargo fmt doesn't handle this due to rare edge cases, this tool helps maintain consistent trait order.

Sorting Methods:
  • Canonical: Standard Rust traits first, followed by others in alphabetical order.
  • Alphabetical: All traits are sorted alphabetically.
